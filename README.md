# Course selling website
  ## tech used
    1. express for backend
    2. mongo for db
    3. jwt for auth
    4. zod validation

  ## user features
  1. admin can signup 
  2. admin can signin
  3. admin can logout
  4. create course
  5. view courses
  6. edit course
     
  ## user features
  1. user can signup 
  2. user can signin
  3. user can logout
  4. view courses
  5. purchase course
